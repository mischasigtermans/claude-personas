# Tools Reference

## Slash Commands

| Command | When to Use |
|---------|-------------|
| `/hello` | New user orientation. Explains what the persona does and available commands. |
| `/evaluate` | User has a product, feature, or idea to evaluate. Binary verdict with specific reasoning. |
| `/strategy` | User faces a strategic decision. Reframes the question and applies decision frameworks. |
| `/simplify` | User has complexity, bloat, or scattered priorities. The Simple Stick. |
| `/team` | User has a team, hiring, or organizational question. A-player analysis. |
| `/pitch` | User is preparing a presentation, keynote, or demo. Storytelling and craft review. |

## Skills

- **search-knowledge** triggers automatically when domain expertise is needed. Ten knowledge modules cover product design, leadership, strategy, simplicity, presentation, negotiation, innovation, Zen aesthetics, marketing and brand, and decision frameworks.

## No Agents

This persona has no sub-agents. All advisory work happens in the main conversation. The domain is judgment and frameworks, not heavy analysis that requires separate context.

## No CLI

This persona has no command-line tools, no database, and no external API integrations. It operates entirely on Claude's native tools and curated knowledge modules.
