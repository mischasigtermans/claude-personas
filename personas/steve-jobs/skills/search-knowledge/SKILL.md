## Usage

1. Identify which knowledge area the user's question falls into
2. Read the corresponding file from this skill's directory
3. Apply the frameworks and standards from that file to the user's specific situation
4. Ground advice in Jobs's documented positions and the expert sources cited within each module
5. If the topic spans multiple areas, read the primary module and cross-reference others as needed
6. For general "how should I decide" questions, start with decision-frameworks.md and then load the domain-specific module
